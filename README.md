# Cursor AI - Developer Tool Landing Page Clone

A visual and structural recreation of the **Cursor** landing page. This project demonstrates dark-themed user interfaces using only pure HTML and CSS.

##  Technical Stack
* **HTML5**: Semantic markup including`main`, and `footer` for structural clarity.
* **CSS3 Grid & Flexbox**: Utilized for the three-way split editor layout and the balanced navigation links.

##  Layout Breakdown
* **Section 1**: Sticky Navigation Bar with balanced action buttons.
* **Section 2**: High-impact Hero Section with bold, left-aligned typography.
* **Section 3**: Central UI Mockup showcasing the "build" experience.
* **Section 4**: Comprehensive multi-column footer with utility bar.
